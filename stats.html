Privacy

Telemetry is optional in the launcher. It is used for public aggregate stats only and can be turned off in Options at any time. The current client sends app version, WoW version, renderer, macOS version, and the configured realmlist value. Public charts count each anonymous install once per value per day, so reopening the launcher does not inflate the visible numbers.
